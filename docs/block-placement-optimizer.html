How it works

Each crop has an optimal PPFD range. The mismatch view colors blocks by how far the actual PPFD deviates from optimal. Green = within range, yellow = slightly off, red = poor fit.

Auto-Optimize assigns blocks to minimize total mismatch across all crops, respecting target block counts. It tries high-demand crops (tomato) in the hottest spots first, then fills outward.
